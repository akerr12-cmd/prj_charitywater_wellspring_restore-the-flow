How to Play

Goal

Restore the community well to 100% by completing pipeline runs in this order: Tool -> Challenge -> Solution -> Impact. Each completed run boosts score, helps more people, and advances restoration progress.

Card Types

Tool

Equipment used to prepare repairs (Rope, Pipe, Wrench, Filter, etc.). Tools often start sequences.

Challenge

Obstacles blocking progress (Drought, Broken Pump, Contamination). These must be cleared with the correct Solution.

Solution

Fixes specific Challenges and advances the restoration effort. Each Challenge has exactly one matching Solution.

Impact

Represents real-world change. Playing Impact cards boosts score, increases “people helped,” and can chain for big bonuses.

Board Layout

  • Pipeline Board — Four slots: Tool, Challenge, Solution, Impact.
  • Challenge Slot — Auto-spawns a new Challenge each round.
  • Hand Row — Your playable cards. Tap a card, then tap a valid slot.
  • Twist Layer — Displays active hazards like locks, theft, or heatwave events.

Placement Rules

Selected cards can only go into valid pipeline slots:

  • Tool Slot accepts only Tool cards.
  • Solution Slot accepts only matching Solution cards for the current Challenge.
  • Impact Slot accepts only Impact cards, after a valid Solution is placed.

When all four slots are filled, the run resolves, rewards are granted, and the board resets for the next challenge.

Clearing Challenges

Every Challenge has one matching Solution. Place the correct Solution to clear the Challenge and keep the run alive.

  • Drought → Emergency Water Delivery
  • Broken Pump → New Pump
  • Contamination → Anti-Contamination Protocol
  • Mudslide → Well Cleaning Crew
  • Heatwave → Solar Panel Repair
  • Rusted Pipe → Pipe Replacement
  • Water Theft → Community Training
  • Supply Delay → Pressure Regulator
  • Low Water Pressure → Flow Stabilizer
  • Blocked Well → Leak Seal Operation
  • Clogged Intake → Intake Repair Team
  • Pipe Burst → High-Pressure Seal

Impact Cards

Impact cards finish the run and represent real-world change. Placing Impact:

  • Grants the biggest score burst of the run.
  • Increases "people helped" in your mission report.
  • Advances overall well restoration progress.

Hand Management

Your hand auto-refills after each placement. Use taps strategically to keep key cards available for matching solutions and twist counters.

Scoring and Progress

Each completed run awards score by card type and adds restoration progress. Keep completing runs until the well reaches 100%.

Twist Mechanics ⚠️

Some Challenges trigger special effects when revealed or moved:

  • Contamination Spread — Shuffles random cards.
  • Pipe Lock — Locks a card for several seconds.
  • Heatwave — Speeds up the timer.
  • Blocked Well — Prevents final restoration until cleared.

Winning the Game

To win, you must:

  • Complete pipeline runs consistently.
  • Clear incoming challenges with matching solutions.
  • Reach 100% Well Restoration.

Reaching 100% triggers the celebration screen, confetti, and your final impact report.
