# PIPELINE RESTORATION SOLITAIRE — DESIGN DOCUMENT
*A hybrid patience game combining FreeCell‑style strategy with layered obstacle clearing.*

---

# 1. CORE FANTASY
You are leading a water‑restoration effort in a community whose pipeline system has collapsed.  
Each card represents a real component of the water‑access journey: tools, challenges, solutions, and impact moments.

Your goal is to strategically untangle the broken pipeline, clear obstacles, uncover the buried well, and restore clean water.

---

# 2. CORE STRUCTURE

## Tableau (Main Play Area)
- 8 columns of cards (like FreeCell), each containing layered cards.
- Some cards are face‑down or locked behind challenges.
- Columns vary in depth, creating a sense of “digging” through the pipeline.

## Water Tanks (Free Cells)
- 4–6 temporary storage slots.
- Used to hold cards while reorganizing sequences.
- Some card types cannot be stored (e.g., certain Challenges).

## Foundations
Four foundation piles represent the restoration process:
1. Tools Foundation  
2. Solutions Foundation  
3. Challenges Cleared Foundation  
4. Impact Foundation  

Completing all four is required to win.

---

# 3. CARD BEHAVIOR

## Tools
- Used to prepare or modify Challenges.
- Can be sequenced together.
- Often required before a Solution can be applied.

## Challenges
- Block progress in a column.
- Some freeze a column until solved.
- Some spread penalties if ignored.
- Cannot always be moved or stored.

## Solutions
- Clear specific Challenges.
- Unlock hidden or face‑down cards.
- Advance the Solutions Foundation.
- Sometimes act as wildcards for Tools.

## Impact Cards
- Provide score multipliers.
- Can be placed in their foundation at any time.
- Some reveal hidden cards beneath them.
- Represent emotional milestones in the restoration journey.

---

# 4. SEQUENCING RULES

## Valid Sequences
- Tool → Tool → Solution  
- Challenge → Solution  
- Tool → Solution → Impact  
- Tool → Challenge → Solution (advanced combo)

## Invalid Sequences
- Challenge → Challenge  
- Impact → Challenge  
- Solution → Tool  
- Impact → Tool  

## Movement Rules
- Cards can be moved between columns if they form a valid sequence.
- Cards can be moved to Water Tanks if allowed by type.
- Only the top card of each column is movable unless unlocked.

---

# 5. LAYER‑CLEARING

## Hidden Layers
- Some cards start face‑down.
- Some are “locked” behind a Challenge.
- Some require a specific Solution to reveal.
- Some are buried and require clearing the entire column above.

## Unlocking Methods
- Apply the correct Solution to a Challenge.
- Use Tools to prepare or weaken a Challenge.
- Use Impact Cards to reveal or clear layers.

This creates a sense of digging deeper into the pipeline to reach the well.

---

# 6. TWIST MECHANICS

## Penalty Cards
- **Contamination Spread**  
  Locks 2 random columns until a Solution is applied.

- **Heatwave**  
  Reduces available moves for 30 seconds.

- **Water Theft**  
  Removes an Impact Card unless countered immediately.

- **Blocked Well**  
  Prevents access to the final foundation until cleared.

## Strategic Twist
- Water Tanks are limited and strategic.
- Some cards degrade if stored too long.
- Some Challenges cannot be stored at all.
- Some Solutions lose power if not used quickly.

---

# 7. WIN CONDITIONS

To win, the player must:

1. Clear all Challenges in the tableau.  
2. Build all four Foundations (Tools, Solutions, Challenges Cleared, Impact).  
3. Uncover the **Well Card** at the bottom of the deepest column.  
4. Place the final Solution on the Well Card.  

This triggers the win celebration.

---

# 8. IMPACT SCREEN

After winning, the player sees:

- Final Score  
- Moves Taken  
- Challenges Cleared  
- Tools Used  
- Solutions Applied  
- Impact Cards Earned  
- “People Helped” (fictional but meaningful metric)  
- Time Spent  
- Return to Main Menu button  

The screen uses Charity: Water branding and emotional microcopy.

---

# 9. FULL RULESET

## Setup
- Shuffle the 52‑card deck.
- Deal 8 tableau columns with varying depths.
- Place 4–6 Water Tanks above the tableau.
- Place 4 empty Foundations to the right.

## Turn Structure
Players may:
- Move a card to another column (if sequence is valid).
- Move a card to a Water Tank (if allowed).
- Move a card to a Foundation (if valid).
- Apply a Solution to a Challenge.
- Use Tools to prepare Challenges.
- Reveal hidden cards when unlocked.

## Restrictions
- Only the top card of a column may be moved unless unlocked.
- Some Challenges cannot be moved or stored.
- Some Solutions expire if not used within a certain number of moves.
- Some Impact Cards can only be played after a Solution is applied.

## Losing Conditions
- No valid moves remain.
- All Water Tanks are full and no sequences can be formed.
- A spreading Challenge (e.g., Contamination) locks too many columns.
